# Business Intelligence & Predictive Analytics – Global Superstore

## Project Overview
This project presents an end-to-end Business Intelligence and Predictive Analytics solution developed using Power BI and R. The objective of the project is to analyse global sales performance, profitability, discount impact, and regional trends to support data-driven business decision-making.

## Dataset
The analysis is based on the Global Superstore dataset obtained from Kaggle. The dataset contains transactional sales data including orders, customers, products, regions, shipping details, discounts, and profit information.

Source: Kaggle – Global Superstore Dataset

## Tools & Technologies
- Power BI
- DAX
- Power Query (M Language)
- R (Machine Learning & Regression Analysis)
- GitHub (Version Control)

## Data Preparation & Modelling
- Performed data cleaning and transformation using Power Query.
- Designed a snowflake-style data model with fact and dimension tables.
- Created calculated columns and measures to support advanced analytics.

## Key Analysis & Features
- Sales and profit trend analysis with KPIs and moving averages.
- Regional and category-level profitability analysis.
- Discount impact analysis on profit margins.
- AI visuals including Key Influencers and Decomposition Tree.
- Predictive profit analysis using machine learning (linear regression) in R.

## Dashboard Pages

- Geographical and Category-Level Insights
- AI-Driven Profitability Drivers & Root Cause Analysis
- Predictive Analytics for Profit Optimisation
